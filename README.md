# Linear Regression Examples

This repository contains three simple linear regression projects implemented as Jupyter notebooks:

- **Car Price Prediction** (`CAR PRICE PREDICTION.ipynb`)
- **House Price Prediction** (`HOUSE PRICE PREDICTION.ipynb`)
- **Student Score Prediction** (`STUDENT SCORE PREDICTION.ipynb`)

Each notebook walks through the full workflow:
1. Import required libraries (`pandas`, `scikit-learn`, `matplotlib`).
2. Load and preprocess a CSV dataset.
3. Define independent and dependent variables.
4. Split data into training and testing sets.
5. Train a `LinearRegression` model.
6. Make predictions and evaluate using R² score.
7. Visualize the regression line.

A consolidated Markdown file (`linear_regression_examples.md`) is also included for reference and easy copying of code snippets.

## Getting Started

1. Clone this repository:
   ```bash
   git clone https://github.com/doyancha/Linear-Regression.git
   ```
2. Open any notebook with Jupyter Lab/Notebook.
3. Install dependencies if needed:
   ```bash
   pip install pandas numpy scikit-learn matplotlib
   ```
4. Run the cells sequentially to reproduce the examples.

Feel free to modify the datasets or expand the models for learning purposes.
